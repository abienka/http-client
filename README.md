## Installation

Use the [composer](https://getcomposer.org/) to install required packages.

```bash
$ composer install
```

## Unit tests

```bash
$ ./vendor/bin/phpunit --testsuite Unit
```

## Functional tests

```bash
$ ./vendor/bin/phpunit --testsuite Functional
```
